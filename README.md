## Project Name
Fingerprint Based Door Lock System

## Project Scope
The scope of this project is to develop a secure door locking system using fingerprint authentication. The system allows access only to authorized users by verifying their fingerprints, enhancing security and eliminating the need for traditional keys. It provides a reliable solution for homes, offices, and restricted areas.

## Project Description
The Fingerprint Based Door Lock System uses a fingerprint sensor connected to an Arduino microcontroller to identify authorized users. When a registered fingerprint is detected, the system activates a servo motor to unlock the door. Push buttons are used for system control and user management functions such as enrollment and reset. If an unauthorized fingerprint is detected, access is denied. This project improves security and offers a convenient keyless entry solution.
